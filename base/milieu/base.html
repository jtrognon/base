différents types d'éléments HTML

block
ex : p, p, liste, forme, div
commence toujours sur nouvelle ligne et prend toute la laregeur
il peut contenir des éléments de types block et inline
inline
ex : strong , em, a, img, span
inline ne commence pas sur une nouvelle ligne et s'insère dans ligne actuelle

chaque éléments HTML est soit de type block soit de type inline

je suis un élément de type block
